# ⚡¿Qué hace un onboarding exitoso?

Un onboarding exitoso es el que logra que el doctor **vea valor real, rápido**. Estos son los pilares:

| **Pilar** | **Qué significa en la práctica** |
|-----------|----------------------------------|
| 🚀 **Empieza antes de la sesión** | Haces análisis previo del cliente para poder ofrecer asesoría personalizada. |
| 🤝 **Genera rapport** | Creas una conexión genuina con el doctor. Es la clave para construir confianza. |
| 🧠 **Eres consultor, no instructor** | No es una clase de cómo usar Doctoralia. Es una consultoría de cómo con Doctoralia el doctor logra sus objetivos. |
| 💡 **Traduces features en valor** | No explicas funciones. Explicas qué problema resuelven y su beneficio. |
| 📈 **Identificas oportunidades** | Detectas señales de necesidades adicionales (Upselling y Cross-Selling) para generar más valor al cliente. |
